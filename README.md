# SANTUM · Libro de Recetas 🌿

App PWA para gestión de recetas cosméticas de SANTUM.

---

## ⚡ Publicar en 15 minutos (gratis)

### Paso 1 — Sube el código a GitHub

1. Ve a **github.com** → Crear cuenta gratis si no tienes
2. Clic en **"New repository"**
3. Nombre: `santum-recetas` → **Create repository**
4. Arrastra la carpeta `santum-app` completa al repositorio
5. Clic en **"Commit changes"**

### Paso 2 — Publica en Netlify

1. Ve a **netlify.com** → Crear cuenta gratis (con tu cuenta de GitHub)
2. Clic en **"Add new site"** → **"Import from Git"**
3. Selecciona el repositorio `santum-recetas`
4. Build command: `npm run build`
5. Publish directory: `build`
6. Clic en **"Deploy site"**
7. En 2 minutos tendrás una URL tipo: `https://santum-recetas.netlify.app`

### Paso 3 — Instalar como app en iPhone/iPad

1. Abre la URL en **Safari**
2. Toca el botón **Compartir** (cuadrado con flecha)
3. Selecciona **"Añadir a pantalla de inicio"**
4. Ponle nombre **SANTUM** → **Añadir**
5. ¡Ya aparece como app en tu pantalla!

### Paso 4 — Instalar en Mac

1. Abre la URL en **Safari** o **Chrome**
2. En Safari: menú **Archivo** → **"Añadir a Dock"**
3. En Chrome: clic en el icono de instalar (⊕) en la barra de direcciones

---

## 🔄 Actualizar la app

Cuando quieras hacer cambios:
1. Modifica los archivos en GitHub
2. Netlify recompila automáticamente en ~2 minutos
3. La app se actualiza sola en todos los dispositivos

---

## 📱 Funciona sin internet

Una vez instalada, la app funciona **offline** gracias al Service Worker.
Los datos se guardan localmente en cada dispositivo.
